GTK 3 equalizer
===============

#### Requirements

The example is built using [libvlc-go](https://github.com/adrg/libvlc-go) and [gotk3](https://github.com/gotk3/gotk3).

#### Build

See build instructions at https://github.com/adrg/libvlc-go/wiki/Build-GTK-3-examples.
